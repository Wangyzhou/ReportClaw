## 关键技术决策

| 决策 | 为什么 |
|------|--------|
| 不接 OpenClaw gateway，用 Python chat server 替代 | OpenClaw v3 pairing 协议复杂，跨机 token 不通 |
| Coordinator V4-Pro / Writer+Reviewer V4-Flash | V4-Pro 在 Writer 场景慢 5-10x（reasoning_tokens），demo 体验差 |
| 价格 hardcode V4-Flash + V4-Pro 真实标价 | OpenClaw 配置是过时 V3 价格 |
| Cost 计算去掉 Claude 对比 | "用 deepseek 就用 deepseek" |
| 引用追踪用 native `title` tooltip | 30 分钟可上，比 React popover 简单 6x |
| 任务树用 Spring Boot `/api/tasks` 真写真读 | 跟王亚洲 PR #4 的 TaskProgressController 兼容 |
